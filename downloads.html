Downloads

Astrid is available as a collection of NuGet packages. There is a package for the core framework and indiviual packages for each platform target. This allows you to choose which targets you want to work with in your projects.

Official Packages

The official package releases are hosted on nuget.org

  • Astrid.Framework - Reference this package from every project in the solution.
  • Astrid.Windows - Reference this package from projects targeting Windows.
  • Astrid.Android - Reference this package from projects targeting Android.

It is recommended that you reference the packages using the NuGet Package Manager in Visual Studio or from the Package Manager Console.

Development Builds

Prerelease alpha builds for every commit are also available from the build server. You can add a new package source in the NuGet Package Manager settings in Visual Studio using the following URL:

http://build.craftworkgames.com/guestAuth/app/nuget/v1/FeedService.svc/

Remember: Select Include Prerelease when referencing the packages.

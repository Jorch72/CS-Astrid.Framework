Astrid Game Framework

Astrid is a C# game-development framework designed to speed up game development of mobile games by allowing most of the development to happen on the desktop using the same code base.

Download Github

Cross Platform

Built from the ground up as a multi-platform architecture. Develop your game on the desktop, then quickly port it to other platforms with only a few lines of code.

Open Source

The source code is hosted on github and licensed under the MIT license. Basically, you can do whatever you want as long as you include the original copyright and licenses in their respective sources.

Portable & Extensible

The core framework is built using Portable Class Libraries (PCL). This makes it easy to write your own extensions without having to worry about compiling it for each supported platform.

Entities & Components

The framework comes with a powerful entity component engine. Compose game objects and scenes using built-in components and systems, or create your own.

Sub-Systems Included

Astrid is programmer friendly, it sits in between a full blown game engine and a low level framework. We've already done the heavy lifting to add physics and particle effects, or you can roll your own if you prefer.

Astrid Studio

Currently in development is a powerful scene editor for creating great looking GUI's and levels for your games.
